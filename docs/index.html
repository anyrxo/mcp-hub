Quick Start

Commands

Powerful commands for every need

mcp search <query>

Search for MCP servers by name or description. Filter by category, verification status, and more.

mcp install <package>

Install an MCP server and automatically configure Claude Code. Handles all the setup for you.

mcp list [--all]

List installed packages or all available packages in the registry.

mcp info <package>

Get detailed information about a package including metadata, installation details, and links.

mcp categories

View all available package categories with counts.

mcp config [--json]

View your current Claude Code MCP configuration.

mcp doctor

Run system diagnostics to ensure everything is working correctly.

mcp uninstall <package>

Remove an installed MCP server and clean up configuration.
